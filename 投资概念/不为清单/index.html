    不为清单（Stop Doing List）

    知道什么不该做，比知道该做什么更重要。不为清单是段永平的经营哲学和投资纪律核心。

    核心思想

    "人们关注我们往往是因为我们做了的那些事情，其实我们之所以成为我们，很大程度上还因为我们不做的那些事情。" — 段永平

    不为清单的本质： - 知道什么不该做比知道该做什么更重要 - 更健康更长久是目标，不为清单是手段 - 避免致命错误，比追求高收益更重要

    段永平的不为清单

    投资领域

    不做 原因
    不用margin（杠杆） 多数企业最后死在资金链断裂上
    不投不懂的公司 超出能力圈就是赌博
    不追涨杀跌 跟风是亏损的根源
    不借钱投资 杠杆会把命运交给市场疯子
    不做短线交易 投机不是投资

    企业经营领域

    不做 原因
    不赚不应该赚的钱 本分文化的核心
    不欺骗消费者 口碑是企业的生命线
    不做损害行业的事 行业健康企业才能长久
    不盲目多元化 专注才能做强
    不做超出能力的扩张 扩张过度会失控

    芒格的智慧

    "如果知道自己会死在哪里就坚决不去那里。" — 芒格

    不为清单的逻辑基础： - 反向思考：不想失败，就想怎样会失败，然后避免 - 不是追求成功，而是避免失败 - 不是追求高收益，而是避免本金损失

    不为清单 vs 正面清单

    维度 不为清单 正面清单
    思维方式 反向思考 正向思考
    关注点 避免错误 追求成功
    适用场景 风险控制 目标设定
    执行难度 相对容易 相对困难
    效果 防守型 进攻型

    最佳实践：两者结合，不为清单保底线，正面清单谋上限。

    如何建立自己的不为清单

    投资者的不为清单模板

    第一层：铁律（绝对不做） - [ ] 不借钱炒股 - [ ] 不用杠杆 - [ ] 不投看不懂的公司 - [ ] 不追涨杀跌 - [ ] 不听消息跟风

    第二层：纪律（尽量不做） - [ ] 不频繁交易 - [ ] 不满仓 - [ ] 不单一持仓 - [ ] 不追热点 - [ ] 不做短期预测

    第三层：原则（原则上不做） - [ ] 不买高估值股票 - [ ] 不买亏损公司 - [ ] 不买管理层有问题的公司 - [ ] 不买行业下行的公司 - [ ] 不买护城河不深的公司

    企业经营者的不为清单模板

    经营铁律： - [ ] 不做虚假宣传 - [ ] 不欺骗消费者 - [ ] 不做损害行业的事 - [ ] 不盲目扩张 - [ ] 不做短期主义决策

    不为清单的威力

    为什么不为清单比正面清单更有效？

    1. 避免致命错误
    2. 一次致命错误可以摧毁多年积累

    3. 不为清单是防守，保住底线

    4. 简化决策

    5. 知道什么不该做，剩下的选择就少

    6. 选择少了，决策更容易

    7. 降低认知负担

    8. 正面清单需要不断判断"够不够好"

    9. 不为清单只需判断"是不是不该做"

    10. 长期复利

    11. 避免错误比追求成功更能积累
    12. 不亏钱是复利的前提
